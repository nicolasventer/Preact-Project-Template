### Qu'est-ce que React Project Template ?

C'est un template dogmatique pour un projet React avec de nombreuses fonctionnalités déjà configurées.  
Il peut être utilisé en tant que template `Full` ou `Light`. *(Version Light en cours)*

La `version Light` retire les dépendances avec `typebox`, `mantine`, `react-hot-toast`, `lucide-react` et `react-icons`.
