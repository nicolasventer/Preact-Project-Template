### What is React Project Template?

It is an opinionated template for a React project with many features already set up.  
It can either be used as a `Full` or `Light` template. *(Light version WIP)*

The `Light version` remove the dependencies with `typebox`, `mantine`, `react-hot-toast`, `lucide-react` and `react-icons`.
